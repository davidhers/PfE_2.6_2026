Overweighntess and SES

This project is about overweight people in the Netherlands. It tries to quantify and show the significance of this problem. It returns several graphs and new data sets. 

The data used for this project was all found on the website of CBS. They consist of a dataset about overweight over time in the Netherlands, average income per province,
the percentages of different education levels per province and lastly the percentage of overweight people per province. These last three dataset are used to try and
compare them, to see if there might be some kind of correlation between these things. 

Using this code is very easy. If all the packages are installed already, run the entire code (the datasets are already downloaded to the map), otherwise install the 
packages and run the entire code.
